# Project Countries

An interactive React app for viewing information about countries around the world.  
Allows you to search for countries, view details, and switch interface themes.

Link: https://project-countries-six.vercel.app/

---

## Functional

- Search for countries by name  
- View a list of countries with flags  
- Country detail page  
- Toggle light/dark theme  
- Google Maps integration  

### Information:
- Population  
- Region  
- Capital  
- Languages  
- Currency  

- Pagination (page switching)
